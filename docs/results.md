# Retina Results

Results are generated locally and written to:

- `reports/retina_retrieval_eval.json`
- `reports/retina_retrieval_eval.md`
- `reports/retina_runtime_benchmark.json`
- `reports/retina_runtime_benchmark.md`
- `reports/retina_retrieval_failures.jsonl`
- `reports/retina_retrieval_failures.md`

If no run has been executed yet, the reports will be absent or marked pending.
